# PocketAI

PocketAI is an open-source Python client that connects to an Ollama server running locally on a phone or computer.

## Features

- 🤖 Chat with local AI models
- 📱 Connect to Ollama running on Android (Termux)
- 💻 Simple command-line interface
- 🆘 `/help` command
- 👋 `/bye` command
- 🔒 Fully local (no cloud required)

## Requirements

- Python 3.13+
- Ollama
- Requests

## Installation

pip install -r requirements.txt
